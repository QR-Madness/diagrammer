## Quick Tips

- **Double-tap a tool key** (like `R` `R`) to stay in that tool mode
- **Press `V`** at any time to return to the Select tool
- **Hold `Space`** temporarily switches to Hand tool for panning
- **Right-click** always opens the context menu with more options

## Customizing Shortcuts

Custom shortcuts can be configured in **Settings → Keyboard Shortcuts**.

1. Open Settings (`Ctrl+,`)
2. Go to **Keyboard Shortcuts**
3. Click a shortcut to edit
4. Press new key combination
5. Click **Save**

### Conflicts

If a shortcut conflicts with an existing one:
- The new shortcut takes precedence
- The old action becomes unbound
- A warning is shown for review

### Reset Defaults

To restore default shortcuts:
1. Go to **Settings → Keyboard Shortcuts**
2. Click **Reset to Defaults**
3. Confirm the reset
