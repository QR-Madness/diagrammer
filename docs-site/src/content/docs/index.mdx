## Features

<CardGrid stagger>
  <Card title="High Performance" icon="rocket">
    Render 10,000+ shapes at 60fps using Canvas 2D API with spatial indexing and viewport culling.
  </Card>
  <Card title="Real-time Collaboration" icon="group">
    Work together with Protected Local mode. CRDT-based sync ensures conflict-free editing.
  </Card>
  <Card title="Rich Shape Libraries" icon="puzzle">
    Built-in flowchart, UML, and ERD shapes. Create and share custom shape libraries.
  </Card>
  <Card title="Desktop & Web" icon="laptop">
    Runs as a native desktop app (Tauri) or in your browser. Full offline support.
  </Card>
  <Card title="Export Anywhere" icon="document">
    Export to PNG, SVG, or JSON. Copy to clipboard. Import from various formats.
  </Card>
  <Card title="Dark Theme" icon="moon">
    Beautiful dark interface with customizable themes and style profiles.
  </Card>
</CardGrid>
